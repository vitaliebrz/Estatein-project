# Estatein-project
This project is a front-end implementation of a real estate website UI design, based on a Figma template. The goal is to convert the provided design into a fully functional website using HTML, CSS, and JavaScript.

📌 About the Project

The website follows a modern, dark-themed aesthetic designed for a real estate business. It includes layouts and components for:

Homepage with hero section

Property listings

Search & filter UI

Property detail pages

Responsive navigation bar and footer

The original UI design was created in Figma and can be accessed here:
🔗 Figma Design:
https://www.figma.com/design/VaorDQ0YosNVJ19ERrYkqs/Real-Estate-Business-Website-UI-Template---Dark-Theme-%7C-Produce-UI--Community-?node-id=139-6238&t=BGV5w1jCnwrtPExB-0

🚀 Tech Stack

HTML5 — Page structure

CSS3 — Styling and responsive layout

JavaScript — Interactivity and dynamic behavior
